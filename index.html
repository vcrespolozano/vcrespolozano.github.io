Sobre mi

¡Hola!, mi nombre es Víctor, soy técnico superior en "Administración de sistemas informáticos en Red" y también en "Desarrollo de aplicaciones multiplataforma". Mi especialidad, tanto por gusto personal como por experiencia laboral es el sector WEB, donde he trabajado más de dos años, además de hacer proyectos de manera individual.

Cuento con conocimientos básicos de SEO gracias a un curso online de Udemy, además de conocimientos sobre la "Google Search Console". He trabajado durante dos años en una empresa, utilizando el framework "Codeigniter" de PHP (Modelo Vista Controlador) y mis conocimientos se centran en el desarrollo WEB "a medida", desarrollando front-end sin el uso de plantillas, todo desde 0 incluyendo maquetación responsive. Dispongo también de experiencia en el desarrollo de back-end, todo ello utilizando PHP como lenguaje, utilizando funciones propias. Respecto a las bases de datos, siempre he trabajado con MySQL, utilizando la herramienta WorkBench de la propia compañía.

Para la maquetación, utilizo HTML5, CSS3 y jQuery, además de Ajax para las interacciones dinámicas con la BD.

Recientemente he adquirido conocimientos sobre SASS y los estoy aplicando en proyectos personales (como esta web por ejemplo), ya que me parece muy potente para desarrollar CSS con herramientas que poseen otros lenguajes de programación.

Como herramientas de trabajo he utilizado Sublime Text y, ahora de manera personal, utilizo Visual Studio Code.

Poseo también una certificación en GitHub, conseguida a traves de Udemy, en la que conocí y aprendí como funcionan los sistemas de control de versiones, además de una de las plataformas más conocidas, GitHub.

Actualmente especializado en la tecnología REACT JS. Con más de 2 años de experiencia profesional.

Tecnologías
PHP Lenguaje "server-side" con el que se construye el esqueleto funcional de la mayoría de sitios Web que he desarrollado. Tanto con el framework "Codeigniter", como utilizando el lenguaje de manera nativa desarrollando mis propias funciones.
HTML Obligatorio y el pilar base para hacer el esqueleto visual tanto del front-end como del back-end.
CSS El lenguaje que le da forma y maquilla nuestro esqueleto HTML. Imprescindible para que un sitio WEB coja su forma visual y obtener una funcionalidad agradable y vistosa para el usuario.
SASS Preprocesador para CSS, que incrementa sus posibilidades a la hora del desarrollo, además de hacerlo más legible y modulable para trabajar con él.
jQuery Otro de los pilares fundamentales en prácticamente cualquier sitio WEB, para otorgarle funcionalidad "en caliente", así como dotar al usuario de interacciones con la página sin necesidad de navegar a otras páginas.
MySQL El lenguaje encargado del almacenamiento de los datos, intermediado a través de PHP para la obtención y el ingreso de datos.
